# Front-end

Instalar o Angular CLI de forma global na sua máquina;
    npm i -g @angular/cli

Inicia o servidor back-end:
    cd backend
    npm start

Iniciar app Angular (inicia na porta padrão 4200);
    cd frontend
    npm start

------------------------
